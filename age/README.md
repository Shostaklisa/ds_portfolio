# Age Estimation from Face Images
**Описание проекта**

В этом проекте решается задача оценки возраста человека по фотографии лица с использованием сверточных нейронных сетей.
Основная цель - проверить, насколько предобученные CNN-модели (в частности, ResNet) устойчивы и эффективны на относительно небольшой выборке изображений.

Проект носит исследовательский характер и фокусируется на анализе данных, выборе архитектуры и интерпретации результатов.

**Данные**

Размер выборки: 7 591 изображение

Изображения лиц разного качества (встречаются размытые фотографии)

Распределение по возрасту несбалансированное: преобладают молодые люди? возрастная группа 60+ представлена слабо.
Это накладывает ограничения на качество предсказаний для пожилых людей и может смещать модель в сторону среднего возраста.

**Подход**

Использована предобученная сверточная нейронная сеть ResNet50. Обучение проводилось в режиме регрессии (предсказание возраста)

Рассматривались:
- обучение на чистых весах vs transfer learning
- влияние качества изображений
- переобучение глубокой модели на небольшой выборке

Для небольшой выборки использование предобученных моделей оказалось более устойчивым решением по сравнению с обучением CNN «с нуля».

**Результаты и анализ**

Модель в целом улавливает зависимость между изображением и возрастом. Средняя ошибка порядка ~6 лет, что для задачи оценки возраста всё ещё считается достаточно высокой. 
Наблюдается переобучение, типичное для глубоких сетей на небольших датасетах. Предсказания для возрастов 60+ менее стабильны из-за дисбаланса данных

*Возможные улучшения*

- Использовать линейную активацию на выходе вместо ReLU
(с последующей постобработкой отрицательных значений)

- Применить двухэтапное обучение:
  Заморозить backbone (trainable = False) и обучить только «голову»
  Частично или полностью разморозить ResNet с очень маленьким lr ≈ 1e-5
-  Добавить аугментации изображений
-  Использовать взвешенную функцию потерь или методы балансировки данных

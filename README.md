# Прогнозирование количества заказов такси на следующий час с использованием временных рядов

В этом проекте реализован анализ и прогнозирование временного ряда для предсказания количества заказов такси на следующий час.

**Основные этапы работы:**
- Подготовка данных
- Импорт библиотек для анализа данных, визуализации и машинного обучения.
- Загрузка и первичная обработка данных: проверка типов, пропусков, ресемплирование по часу с подсчётом количества заказов.
- Анализ данных (EDA)
- Построение описательной статистики, гистограмм и боксплотов.
- Определение выбросов и шумов с помощью анализа тренда, сезонности и остатков временного ряда.
- Выявление тенденций и циклов, анализ тренда и сезонности.
- Создание признаков для моделей
**Временные признаки**: час, день, день недели.
  
**Лаговые признаки**: количество заказов в предыдущий час (lag_1), сутки назад (lag_24).

**Скользящее среднее для краткосрочных трендов** (окно = 4).

**Циклическая кодировка часового и дневного признаков для нейросети**.

- Анализ корреляции признаков с помощью phik-матрицы.
- Обучение моделей и прогнозирование
  
**Использованы модели**: CatBoost, LightGBM, полносвязная нейросеть (MLP).

Выделена тестовая выборка (10% данных) с shuffle=False.

Подбор гиперпараметров и кросс-валидация на 3 фолдах.

Выбор лучшей модели по RMSE (≤48) и расчёт прогноза на тестовой выборке.

## Заключение:
В результате работы создана модель, способная прогнозировать количество заказов такси на следующий час с высокой точностью, что позволяет планировать ресурсы и повышать эффективность работы сервиса.

**Ключевые технологии**:
*Time series, CatBoost, LightGBM, Neural Network (MLP), Python, phik, scikit-learn, PyTorch*.

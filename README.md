# Краткое описание проектов

| № | Название проекта | Описание | Стек |
|---|------------------|----------|------|
|1|**forecast-purchasing**|Проект посвящён прогнозу снижения покупательской активности постоянных клиентов интернет-магазина. На основе данных о покупках, поведении на сайте и взаимодействии с компанией была построена модель, предсказывающая вероятность падения активности в ближайшие три месяца. Проведена предобработка данных, подбор моделей и анализ важности признаков. Дополнительно выполнена сегментация клиентов для последующей персонализации предложений.|*Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly, Phik, Scipy, Shap, Scikit-learn (предобработка, Pipeline, модели: DecisionTree, KNN, SVC, LogisticRegression, подбор гиперпараметров)*|
|2|**image-text-matching**|Модель определяет, насколько изображение соответствует текстовому описанию. Для этого использованы эмбеддинги BERT для текста и ResNet-18 для изображений, объединённые и поданные в MLP. Реализована фильтрация нежелательного контента и дисклеймер для изображений детей. Лучшая модель достигла RMSE ≈ 0.21.|*Python, Pandas, NumPy, Matplotlib, Pillow, DeepFace, PyTorch, Torchvision, BERT (Transformers), scikit-learn, SciPy, tqdm*|
|3|**PySpark Housing**|Проект посвящён предсказанию медианной стоимости жилья в Калифорнии на основе данных 1990 года. Были обучены две модели линейной регрессии: одна с полным набором признаков (включая категориальный ocean_proximity), другая только с числовыми признаками. В ходе работы восстановлены пропуски, проведена предобработка данных. Результаты показали, что добавление признака близости к океану слегка улучшает метрики, но обе модели имеют умеренную точность.|*Python, Pandas, NumPy, Matplotlib, Seaborn, PySpark (SparkSession, DataFrame, SQL functions), PySpark ML (StringIndexer, OneHotEncoder, VectorAssembler, StandardScaler, LinearRegression, RegressionEvaluator, ParamGridBuilder, CrossValidator), Logging*|
|4|**steel-thermal-model**|Проект прогнозирования температуры стального сплава на металлургическом комбинате с целью оптимизации производственных расходов. Данные из нескольких таблиц Sqlite были объединены и предобработаны, проведён EDA и корреляционный анализ. Лучшая модель - XGBoost, показала MAE 6.2, RMSE 8.3, R² 0.46, MAPE 0.39% и может использоваться для имитации технологического процесса и поддержки решений по оптимизации расходов.|*Python, Pandas, NumPy, Matplotlib, Seaborn, SciPy, Phik, SQLAlchemy, scikit-learn (RandomForest, Ridge, DummyRegressor, cross-validation, preprocessing, pipeline, metrics, permutation importance), XGBoost, LightGBM, TensorFlow / Keras (Sequential, Dense, Dropout, Adam, EarlyStopping, KerasRegressor)*|
|5|**stellar-temperature-model**|С помощью нейронной сети предсказываем температуру поверхности звезд. Проведена предобработка, исследовательский анализ, нормализация числовых признаков и кодирование категориальных. Построена baseline MLP с двумя скрытыми слоями и кросс-валидацией. Улучшение модели с подбором гиперпараметров слегка повысило точность. Модель хорошо предсказывает температуру распространённых звезд, но недостаточно точна для редких и очень горячих объектов.|*Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly, PyTorch (nn, optim, DataLoader, TensorDataset), Skorch (NeuralNetRegressor), scikit-learn (Pipeline, ColumnTransformer, OneHotEncoder, StandardScaler, GridSearchCV, KFold, train_test_split, metrics), Phik*|
|6|**time_series_taxi**|Проект посвящён прогнозированию количества заказов такси на следующий час на основе временного ряда. Проведена предобработка данных, ресемплирование по часу, добавлены временные признаки и лаговые функции. Были обучены модели CatBoost, LightGBM и полносвязная нейросеть с кросс-валидацией. Лучшие результаты показала нейросеть, модель может использоваться для планирования загрузки водителей.|*Python, Pandas, NumPy, Matplotlib, Seaborn, Statsmodels (seasonal_decompose), Phik, scikit-learn (Pipeline, ColumnTransformer, TransformedTargetRegressor, RandomizedSearchCV, TimeSeriesSplit, train_test_split, preprocessing, metrics), CatBoost, LightGBM, PyTorch (nn, functional, DataLoader, TensorDataset), Skorch (NeuralNetRegressor)*|

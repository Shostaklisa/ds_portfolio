# Модель прогнозирования температуры стального сплава
## Описание проекта
Этот проект посвящён разработке модели машинного обучения для прогнозирования температуры стального сплава на выходе технологического процесса плавки. Разработанная модель используется для снижения энергопотребления на этапе обработки стали и имитации технологического процесса.

## Цель проекта

- Оптимизация энергопотребления при обработке стали.

- Построение модели для определения температуры стального сплава.

- Поддержка моделирования и анализа технологического процесса.

## Данные

Исходные данные были предоставлены в виде нескольких таблиц в базе данных SQLite. Каждая таблица содержала информацию об отдельных этапах технологического процесса.

## Выполненные шаги:

1) Подключение к базе данных SQLite.

2) Проверка наличия и корректности таблиц.

3) Загрузка данных в pandas DataFrame для дальнейшей работы.

4) Предобработка данных

5) Приведение названий колонок к формату snake_case.

6) Поиск и удаление полных дубликатов.

7) Анализ таблиц по отдельности (размер, пропуски, типы данных).

8) Приведение типов данных.

9) Генерация дополнительных признаков, потенциально полезных для анализа и модели.

10) Исследовательский анализ данных (EDA)

**Для каждой таблицы были выполнены:**

11) Рассчёт описательной статистики.

12) Обработка пропусков (заполнение или удаление).

13) Визуализация распределений (гистограммы, boxplot).

14) Оценка выбросов на основе статистики и технологических норм.

15) Объединение таблиц

**Таблицы были объединены с учётом следующих условий:**

- Каждой партии должен соответствовать один объект.

- Во всех партиях присутствовали сыпучие и проволочные добавки.

- Всегда выполнялась продувка газом и нагрев.

16) Были сформированы агрегированные признаки по материалам и параметрам процесса.

17) Отбор признаков

18) Проведён корреляционный анализ.

19) Удалены мультиколлинеарные признаки.

20) Исключены признаки с нулевой корреляцией.

21) Baseline-модель RandomForestRegressor использовалась для оценки важности признаков.

22) **Моделирование**

Были обучены и сравнивались следующие модели:

- Линейная регрессия (Ridge)

- RandomForestRegressor

- XGBoost

- LightGBM

- Нейронная сеть (Keras)

- DummyRegressor для сравнения

**Критерии выбора модели**

Основная метрика: MAE.

Дополнительные метрики: RMSE, R², MAPE.

Оценка проводилась с 3-фолд кросс-валидацией.

23) **Выбор лучшей модели**

Лучшие результаты показала модель XGBoost.

Результаты на обучении:

MAE: 5.78

RMSE: 7.82

R²: 0.50

MAPE: 0.36%

24) **Результаты на тесте**:

MAE: 6.2

RMSE: 8.3

R²: 0.46

MAPE: 0.39%

25) **Анализ остатков**

Были построены:

Гистограмма распределения остатков

Scatter plot «остатки / предсказания»

Scatter plot «остатки / истинные значения»

Q–Q plot

График «фактическая / предсказанная температура»

Наблюдение: модель слегка недооценивает большие температуры и переоценивает малые — типичный эффект сглаживания.

## Выводы

Разработана стабильная и точная модель прогнозирования температуры стального сплава. Модель XGBoost подходит для использования в имитации технологического процесса и в оптимизации энергозатрат.

## Используемые технологии

Python, pandas, numpy, matplotlib, seaborn, scikit-learn, XGBoost, LightGBM, Keras, SQLite

# Stellar Temperature Prediction with Neural Networks

**Проект:** Определение температуры поверхности звезд с помощью нейросети  
**Цель:** Разработать нейронную сеть для предсказания абсолютной температуры на поверхности звезд, используя данные 240 изученных объектов.  

## Описание проекта

Традиционные методы определения температуры звезды включают:

- Закон смещения Вина  
- Закон Стефана-Больцмана  
- Спектральный анализ  

Эти методы имеют свои ограничения и требуют сложных расчетов. Проект направлен на исследование возможности более точного и удобного определения температуры с помощью машинного обучения.

---

## Данные

Используемый датасет: `/datasets/6_class.csv`  
Содержит следующие характеристики звезд:

| Параметр | Описание |
|----------|----------|
| `luminosity_(l/lo)` | Относительная светимость L/L☉ |
| `radius_(r/ro)` | Относительный радиус R/R☉ |
| `absolute_magnitude_(mv)` | Абсолютная звездная величина |
| `star_color` | Цвет звезды (white, red, blue, yellow, yellow-orange и др.) |
| `star_type` | Тип звезды (0: коричневый карлик, 1: красный карлик, 2: белый карлик, 3: звезды главной последовательности, 4: сверхгигант, 5: гипергигант) |
| `temperature_(k)` | Абсолютная температура на поверхности звезды (K) |

## Этапы работы

1. **Загрузка данных и библиотек**  
   - Импорт необходимых библиотек Python (pandas, numpy, matplotlib, seaborn, sklearn, tensorflow/keras).  
   - Загрузка CSV-файла с данными.  

2. **Предобработка данных**  
   - Приведение названий столбцов к удобному формату.  
   - Преобразование типов данных.  
   - Поиск и удаление дубликатов.  
   - Исправление опечаток и разночтений.  
   - Оценка и заполнение пропусков.  

3. **Исследовательский анализ данных (EDA)**  
   - Описательная статистика числовых признаков.  
   - Визуализация распределений числовых и категориальных признаков.  
   - Поиск выбросов и аномалий.  
   - Выводы по промежуточным наблюдениям.  

4. **Подготовка данных для модели**  
   - Корректировка данных на основе EDA.  
   - Категоризация признаков при необходимости.  
   - Разделение на обучающую и тестовую выборки.  
   - Масштабирование числовых признаков.  

5. **Построение моделей**  
   - Создание baseline модели нейронной сети.  
   - Улучшение модели для повышения точности предсказаний.  

6. **Выводы и результаты**  
   - Сравнение разных моделей.  
   - Оценка точности и удобства использования модели.
